# CRM Frontend

A modern, full-featured Customer Relationship Management (CRM) system designed to help businesses manage their customer interactions, sales pipelines, and team collaboration effectively. Built with React 18 and Material-UI v6, this application provides a clean, responsive interface for tracking contacts, leads, and business opportunities through customizable sales pipelines.

The system supports role-based access control, allowing organizations to define permissions for Admins, Managers, and Sales Representatives. With features like contact assignment, lead tracking, and opportunity management, teams can streamline their sales processes and improve customer relationships.

## Features

- **Dashboard** - Overview of users, contacts, leads, and opportunities
- **Contact Management** - Create, edit, delete, and assign contacts
- **Lead Management** - Track and manage potential customers
- **Pipeline Management** - Manage sales pipelines and stages
- **Opportunity Tracking** - Monitor and manage business deals
- **Role-Based Access** - Admin, Manager, and SalesRep roles with different permissions

## Tech Stack

- **React 18** - Frontend framework
- **Material-UI (MUI) v6** - UI component library
- **React Router v7** - Client-side routing
- **Vite** - Build tool and dev server
- **TailwindCSS** - Utility-first CSS framework
- **Material React Table** - Data table component
- **React Toastify** - Toast notifications

## Prerequisites

- Node.js (v16 or higher)
